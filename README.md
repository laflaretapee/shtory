# shtory

Лендинг на React для компании по пошиву, продаже и установке штор в Раевке / Раевском.

## Стек

- Vite
- React
- Vitest
- GitHub Pages

## Команды

```bash
npm install
npm run dev
npm test
npm run build
```

## Деплой

После пуша в `main` сайт автоматически публикуется через GitHub Actions на GitHub Pages.
